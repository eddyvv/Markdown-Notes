# 信息搜索技巧

## 善用搜索引擎

例如：微信搜一搜

## 善于搜索语法

### 限定关键词包含

```bash
"搜索词"
```

### 搜标题包含

```bash
intitle:搜索词
```

### 限定多个标题关键词包含

```bash
allintitle:搜索词1 搜索词2
```

### 限定文章内容包含

```text
intext:搜索词
```

### 限定网址关键词包含

```bash
inurl:cctv
```

### 限定网址来源

```bash
site:cctv.com
```

### 限定图片尺寸

```bash
imagesize:1920x1080
```

### 限定文件格式

```bash
filetype:pdf
```
