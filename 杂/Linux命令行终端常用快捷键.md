# Linux命令行终端常用快捷键

| 快捷键       | 功能                         |
| ------------ | ---------------------------- |
| Ctrl+e       | 光标移动至最末尾             |
| Ctrl+a       | 光标移动至开始位置           |
| Ctrl+u       | 剪切整个命令                 |
| Ctrl+k       | 剪切此处至末尾的所有内容     |
| Ctrl+d       | 删除光标位置字符             |
| Ctrl+h       | 删除当前字符之前的一个字符   |
| Ctrl+w       | 删除光标位置的前一个单词     |
| Ctrl+Shift+n | 打开一个新终端               |
| Ctrl+y       | 粘贴Ctrl+u或ctrl+k剪切的内容 |
