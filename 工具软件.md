## cheat.sh/cht.sh

安装使用

```bash
curl -s https://cht.sh/:cht.sh | sudo tee /usr/local/bin/cht.sh && sudo chmod +x /usr/local/bin/cht.sh
```

在线使用

```bash
curl cht.sh/tar
```

您现在可以使用以下命令启动 cht.sh：

```sql
cht.sh --shell
```
